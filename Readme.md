# 图像算法服务部署框架

**欢迎白嫖。**

## 0x01 背景

为了简化小团队的在开发出web服务后需要进行的复杂的服务部署，并且帮助小团队进行模块化流程化开发。

**顺便内卷（手动狗头**

## 0x02 特性

1. 可伸缩性
2. 服务解耦
3. 支持传统服务和深度学习的模型的配合
4. 支持流水线服务

## 0x03 使用的第三方库

1. celery
2. redis
3. fastapi
4. ncnn
5. triton
